Introduction

Thanks for checking out HistoBreaker!

The concept of this game is that a Breakout level is constructed from the histogram of an input image. Every block you destroy modifies the original image's histogram, and that change is applied to the image. The game is made in HTML5. I chose HTML5 to investigate if it was an appropriate replacement for Flash, and also to discover how image processing could be applied using Javascript.

To start playing right away, click on any of the "Play Now" links to load up the appropriate image, or paste the URL to a different image in the text box and click the "Custom URL" button. If you want to learn more about the project, check out the "Further Reading" section."

Requirements

The most important requirement of HistoBreaker is an HTML5-compatible standards-compliant browser. I recommend Google Chrome, as this is what the game has been tested in most extensively. However, Firefox and Safari should also work.

It is also strongly recommended that you use a computer and a browser with a resolution of at least 1000x1000. The game will still display on lower resolutions, but you'll be forced to scroll the page in order to see what you're doing.
